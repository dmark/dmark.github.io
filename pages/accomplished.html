Johns Hopkins Data Science Specialization

A 9-part series offered by Johns Hopkins University and hosted at Coursera. This MOOC series uses R for programming assignments.

R Programming

  1. Importing data from disk, manipulating it, producing data frames.
  2. Caching computational results.

Getting and Cleaning Data

Course project: Given a set of training data and test data, produce an R script that:

  1. Merges the training and the test sets to create one data set.
  2. Extracts only the measurements on the mean and standard deviation for each measurement.
  3. Uses descriptive activity names to name the activities in the data set
  4. Appropriately labels the data set with descriptive variable names.
  5. Creates a second, independent tidy data set with the average of each variable for each activity and each subject.

Exploratory Data Analysis

Focus on producing charts of specific features in a data set in R for visual data analysis.

Future Courses

I will be continuing the 9-course series over the next 9 - 12 months. The program culminates in a real-world project working with an as yet un-named third party (possibly SwiftKey, developer of virtual keyboards for smart phones).

University of Washington Introduction to Data Science

This 2-month MOOC from UW uses Python for its programming assignments.

Twitter Sentiment Analysis

  1. Interface with Twitter API to download tweets.
  2. Perform basic sentiment analysis on downloaded data.
  3. Calculate sentiment of unrecognized words based on overall tweet sentiment scores.
  4. Sentiment term frequency analysis.
  5. Determine happiest US state based on sentiment analysis & tweet location.
  6. Find top ten hashtags in twitter stream.

Databases

  1. Basic relational algebra: various SQL queries executed against SQLite3 databases.
  2. Implementing matrix multiplication in SQL.
  3. Calculate document similarity based on term frequency analysis, produce document similarity matrix (table).
  4. Keyword search for “best” matching document based on similarity matrix.

Algorithms in MapReduce

Using Python’s MapReduce framework:

  1. Create an inverted index.
  2. Implement a relational join as a MapReduce query.
  3. Given a social network dataset of (person, friend) pairs, describe a MapReduce algorithm to count the number of friends for each person.
  4. Extending from 3 above, implement a MapReduce algorithm to list all non-symmetric friend relationships.
  5. Given a set of key-value pairs, implement a MapReduce algorithm to remove the last 10 characters from the value in each pair.
  6. Given two matrices in sparse matrix form, implement a MapReduce algorithm to perform matrix multiplication on these matrices.
